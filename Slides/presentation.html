Partie 1 - Présentation de WebGL

Introduction - Partie 1
Historique

  • Fin 1990 - VRML (Virtual Reality Modelling Language)
    • Rendu 3D d'images
    • Interaction entre l'animation et l'utilisateur
    • Syntaxe de type XML
    • Défaut - Demande exigente en bande passante
  • En 2005 - X3D
    • Remplace VRML
    • Standard Web3D
  • Fin 2009 - WebGL

Introduction - Partie 1
Qu'est ce que c'est ?

  • Fondé par Mozilla et le groupe Khronos
  • Permet de gérer dynamiquement des éléments 3d sur le web
  • Standards d'OpenGL
  • Utilise le GPU de l'ordinateur
  • API Javascript
  • Utilise l'élément canvas de HTML
  • Supporté par les navigateurs modernes et mobiles

TP Partie 1 - Cours
Le moteur de rendu (Renderer)

  • Affiche les scènes créées
  • Doit être intégré dans une balise div de notre fichier Html
  • Attribut setSize() pour spécifier la taille de la fenêtre de rendu
  • Plusieurs type de Renderer avec Three.js
    • THREE.WebGLRenderer()
    • THREE.CanvasRenderer()

TP Partie 1 - Cours
La scène

  • Contient l'ensemble des éléments 3D
  • Est affichée par le biais de notre renderer
  • THREE.Scene()

TP Partie 1 - Cours
Ajouter un renderer et la scène

var renderer, scene;
function init(){
  //On recupère la div du HTML
  var c = document.getElementById("gameCanvas");

  //On initialise le moteur de rendu
  renderer = new THREE.WebGLRenderer();

  //Taille de la scene
  renderer.setSize(window.innerWidth, window.innerHeight);

  //On ajoute au HTML
  c.appendChild(renderer.domElement);

  //On initialise la scène
  scene = new THREE.Scene();
}

TP Partie 1 - Cours
La caméra

  • Permet de visualiser les éléments de la scène d'un certain point de vue
  • Constructeur THREE.PerspectiveCamera
    • Fov (Field Of View) - correspond à l'angle de vision
    • Aspect - ratio largeur/hauteur
    • Near - distance minimale visible par la caméra
    • Far - distance maximale visible par la caméra

TP Partie 1 - Cours
Greffer une caméra

function init(){
    var WIDTH = window.innerWidth, HEIGHT = window.innerHeight;

    //Attributs de la camera
    var VIEW_ANGLE = 90,
    ASPECT = WIDTH / HEIGHT;
    
    camera = new THREE.PerspectiveCamera(VIEW_ANGLE, ASPECT, NEAR, FAR);

    //On ajoute la caméra à la scène    
    scene.add(camera);
}

function draw(){
  renderer.render(scene, camera);
}

TP Partie 1 - Cours
La lumière

  • Attributs principaux
    • Hex - Couleur hexadécimale de la lumière
    • Intensity - puissance de la lumière
    • distance - distance de la lumière par rapport au centre de celle-ci
  • Types
    • THREE.PointLight - Point lumineux multidirectionel
    • THREE.SpotLight - Spot lumineux
    • THREE.DirectionalLight - Lumière directionelle
    • THREE.AmbientLight - Lumière identique sur tous les objets

Partie 3 - Animation et logique du jeu

TP Partie 3 - Cours
Animer des modèles 3D

Tout objet peut être animé

  • Formes géométriques (Plier, tordre, transformer)
  • Textures (Déplacer, redimensionner, tourner, faire défiler)
  • Matériaux (Changement des couleurs, ajout de reflets, modification de la transparence)
  • Lumières (Changement de couleur, mouvements ou clignotements)
  • Caméras (Tourner, déplacer)

TP Partie 3 - Cours
Idée générale

  • Génération des objets
  • Création d'une fonction qui gère l'animation d'un ou plusieurs objets
  • Appel en continu de la fonction d'animation
  • requestAnimationFrame()
